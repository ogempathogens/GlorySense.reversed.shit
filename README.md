# Reversal of Counter-Strike 2
Everything related to reversing CS2 functions. <br>
Repo is not for spoonfeeding/pasting, it's meant to provide useful info for people who know how to use it.

### Structure:
- Folders correspond to modules (example: client.dll)
- Files inside the folders correspond to in-game functions (example: RunCommand), everything about the function itself will be inside the file.
